## Distribution Notes

- PySide6 is used as an unmodified dynamically linked library. No Qt/PySide6
  source is included or modified in this repository.
- Safe Start is not available from PyPI as of this check. The explicit installer
  and build extra therefore use the verified public repository revision above;
  a local checkout is used only through `CARECENTER_SAFE_START_SOURCE`.
- PyInstaller's special exception allows distributing the generated executable
  under the application's own MIT license.
- Binary release preparation must inspect the actual bundled distributions and
  retain their shipped license files.
